## Description

This application is a user management system that uses database sharding and caching to increase performance.
Additionally, it supports receiving messages to update users' profiles

## Instalation

```bash

# Running Docker

$ Docker-compose up -d

# Instaling dependencies

$ npm install

# Running application

$ npm run start:dev

```

## Routes

```bash

$ /users -> Post

```
